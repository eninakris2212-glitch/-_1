Парсинг сайта Киноафиша
Постановка задачи
Проект предназначен для парсинга рейтинга лучших сериалов с сайта Киноафиши (https://www.kinoafisha.info/rating/series).
Основная цель — собрать топ-1000 сериалов, классифицированных по тегу «Сериал», и сохранить информацию в виде таблицы, содержащей следующие поля:
Название сериала
Жанр
Год выпуска
Рейтинг
Инструкция по сборке и запуску
Требования
Для запуска скрипта необходимы следующие библиотеки Python:
pip install requests beautifulsoup4 ixml pandas openpyxl
В файле "Код проекта по парсингу"  Скрипт автоматически выполнит запросы на сайт, соберёт необходимую информацию и сохранит её в таблице Excel формата .xlsx.
