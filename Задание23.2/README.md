Парсинг сайта Киноафиша
Целью данного проекта является создание универсального инструмента парсинга информации о фильмах/сериалах с различных онлайн-сервисов для кинолюбителей. Проект нацелен на решение следующих задач:

Сбор необходимых метаданных о фильмах и сериалах (название, жанры, рейтинги, оценки пользователей);
Обеспечение возможности последующего расширения функционала для построения рекомендаций на основе анализа большого объема данных;
Повышение точности и полноты полученных данных путем параллельного извлечения информации сразу из нескольких источников.
Обоснование: Наличие широкого набора исходных данных позволит создать систему рекомендаций, учитывающую предпочтения множества пользователей и одновременно охватывающую различные сервисы. Такой инструмент сможет стать основой улучшенной рекомендательной платформы, предоставляя пользователям персонализированные подборки контента.
